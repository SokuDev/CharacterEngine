= Introduction =
NextSoku is a framework for applying gameplay patches to Touhou Hisoutensoku.

After its release, Hisoutensoku gained a big following, but sadly the original developers shortly stopped providing patches. Since version 1.10, released in 2009, the game had no updates.

NextSoku is an attempt to finally provide such an update, and better yet, it lets players make their own dream version of Soku.

Unlike SokuPlus (a previous attempt at modding Soku from a different team), NextSoku makes use of an editable configuration system, allowing players to fine-tune game settings, as well enable/disable modifications at will, and create their own.

The complete pool of all available changes is referred to as a modset. The default modset is also (somewhat confusingly) called NextSoku. At the moment there is another custom modset known as Soku Unlimited.

The general goals of NextSoku:
- make the game feel more fair and less frustrating
- improve balance between characters in the game
- create more variety by improving or reinventing underused skills and spellcards
- make gameplay more consistent by fixing long-standing bugs and oversights

Some modifications have heavier implications than others, and some are experimental and might hurt competetive gameplay. In order to have a better degree of control, NextSoku game modifications are ordered into several "tiers" or "levels" that build upon each other. Each level inherits all changes of the previous level, unless marked exclusive for a certain level.


* '''NextSoku Level 0 "Heaven Route"'''
: Minimal set of changes to fix some issues in 1.10 while mostly retaining the same gameplay.

* '''NextSoku Level 1 "Harmony Route"'''
: Similar in spirit to Level 0, but has more significant changes, characters generally have stronger options, which might make gameplay feel more competetive.

* '''NextSoku Level 2 "Hell Route"'''
: Extended combo options and experimental changes. Characters can deal more damage in more ways, as well as have access to stronger pressure options, making the game feel more fast-paced and colorful, but balance might suffer.

One last thing: NextSoku is a work in progress! If something that you find important is missing, it is likely because we couldn't do it yet.

= Installation and running =
1. Make sure you have SWRS toys installed.
2. Extract the contents of the folder into your soku directory.
3. Use NextSokuLauncher.exe to run the mod of your choice.

Enjoy NextSoku and Soku Unlimited!
